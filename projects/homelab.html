My Homelab

No backups... no redundant servers... what more could you ask for?
This is my homelab that host all the services I do and don't need whatsoever.

My "Servers":
• Dell OptiPlex 390 running Debian 13 with Docker
• Dell Wyse 5070 running Windows 10 LTSC
• Dell XPS 17 L702X running Windows Server 2022
• Dell Chromebook 3100 running Debian 12 Bookworm
• Raspberry Pi 4B running Raspberry Pi OS Lite

And more to come!

I have a lot of services/scripts running at all times, each of which requiring different operating systems to run.
For one, some of my projects like Archive944 and 247RetroWeather requires the use of Windows because those two use OBS's "portable" mode that doesn't exist on the Linux equivalents.

That's why my main "server," the OptiPlex 390, runs Debian 13. With Docker, I host the majority of my services on this PC. This includes Pi-Hole, a local Jellyfin server, a Minecraft server, Home Assistant, and Frigate (for home security).

And you can't forget about my PBX server! I have FreePBX running on the Chromebook 3100, set up with some SIP VoIP phones.
